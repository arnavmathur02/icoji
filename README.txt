STRUCTURE:
icoji has 'css' folder which contains the stylesheets, 'font' folder which contains the font files, a 'config.json' configuration file, and this 'README.txt' file which also contains the license information.

USAGE:
Include the styelsheet: '<link rel="stylesheet" href="css/icoji.css">' and then to use a specific icon, use its class name. For example: '<i class="icoji-heart"></i>' Please see the below list for the current available classes.
